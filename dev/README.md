# Devices
This folder contains the source code for all supported devices and peripherals.

# List of devices
| Folder | Description          |
|--------|----------------------|
| `ahci` | AHCI controllers     |
| `char` | IWE CHAR module      |
| `dma`  | Direct memory access |
| `fp`   | Floating point unit  |
| `ide`  | IDE controllers      |
| `scsi` | SCSI controllers     |
| `tty`  | TTY devices          |
